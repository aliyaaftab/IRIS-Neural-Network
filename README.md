# 🌸 IRIS Flower Classification using Neural Network

## What is this project?
I built a neural network that looks at real flower 
measurements and predicts which type of flower it is.

## Flower Types
- Setosa
- Versicolor  
- Virginica

## Dataset
- 150 real flower samples downloaded from Kaggle
- 4 measurements used: sepal length, sepal width, 
  petal length, petal width

## Tools I Used
- Python
- TensorFlow and Keras
- Scikit-learn
- Matplotlib
- Google Colab (free online coding platform)

## Results
- Test Accuracy: 84%
- Training Epochs: 20

## How to Run This Project
1. Download the Iris.csv file from this repo
2. Open Google Colab (colab.research.google.com)
3. Upload the iris_notebook.ipynb file
4. Upload Iris.csv when asked
5. Click Runtime then Run All
